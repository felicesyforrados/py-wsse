=======
py-wsse
=======

`WS-Security`_ (SOAP WSSE) support for Python,
plugin.

``py-wsse`` supports Python 2.7, 3.3, and 3.4.

.. _WS-Security: https://www.oasis-open.org/committees/download.php/16790/wss-v1.1-spec-os-SOAPMessageSecurity.pdf
